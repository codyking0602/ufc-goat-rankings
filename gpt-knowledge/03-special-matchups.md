## Khabib Nurmagomedov vs Islam Makhachev

Core debate: clean dominance vs fuller current championship work.

Khabib's best argument:
- Undefeated UFC run.
- Cleaner peak.
- No UFC loss penalty.
- One of the clearest "nobody solved him" cases ever.
- The McGregor, Poirier, and Gaethje finish stretch creates the strongest lightweight aura argument.

Islam's best argument:
- More current title-fight volume.
- Fuller championship case.
- Modern lightweight depth.
- Adds elite title-level wins and has passed Khabib on the current board.

Default verdict direction:
- Better peak / cleaner dominance: Khabib Nurmagomedov.
- Better championship case: Islam Makhachev.
- Greater current overall case: Islam Makhachev, barely.

Important style note:
Do not explain this with raw category point totals. Use Khabib's undefeated run and Islam's title-fight volume as the swing points.
