# Opponent Quality Top-Five Consistency Audit

Last updated: 2026-07-04

Purpose: resolve the consistency issue Cody caught in Batch 6: I was using 0.75 too often for fighters who were actually clear top-five or true title-level contenders on fight night.

No live app score changes are approved from this worksheet. This file is the review layer before building any live OQ correction module.

## Rule clarified by Cody

```text
Clear top-five or true title-level contender on fight night = usually 1.00.
```

Use 0.75 only when the win is below true elite/title-level on fight night, such as:

- strong ranked contender but not clearly top-five/title-level
- former champion but no longer near elite form
- famous name/rivalry value without current elite status
- weird context that keeps the win below full elite credit
- title-fight participant whose actual opponent quality is not elite enough for 1.00

## Key correction

The mistake was not that every 0.75 is wrong. The mistake was treating some top-five/title-level wins as "support-plus" instead of full elite OQ.
