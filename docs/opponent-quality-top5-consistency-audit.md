# Opponent Quality Top-Five Consistency Audit

Last updated: 2026-07-04

Purpose: audit every prior Opponent Quality batch for the mistake Cody caught in Batch 6: using 0.75 for fighters who were likely clear top-five or true title-level contenders on fight night.

No live app score changes are approved from this worksheet. This is a review layer before building any live OQ correction module.

## Rule clarified by Cody

```text
Clear top-five or true title-level contender on fight night = usually 1.00.
```

Use 0.75 only when the win is below true elite/title-level on fight night, such as:

- strong ranked contender but not clearly top-five/title-level
- former champion but no longer near elite form
- famous name/rivalry value without current elite status
- weird context that keeps the win below full elite credit
- title-fight participant whose actual opponent quality is not elite enough for 1.00

## Why this audit exists

I had been using 0.75 too broadly for some "very strong contender" wins. That was too conservative when the opponent was actually top-five or title-level on fight night.

This file separates:

1. **Already corrected / likely fine**
2. **Needs top-five verification**
3. **Should likely upgrade to 1.00 if verified**
4. **Should stay 0.75 despite name/title context**
