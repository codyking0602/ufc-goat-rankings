Privacy Policy

Last updated: July 6, 2026

Octagon Verdict is a UFC fighter comparison experience that uses a public ranking data feed hosted on this site. This privacy policy explains what information is collected by this site and by the public data feed used for GPT Actions.

Information We Collect

This site does not collect personal information from visitors. It does not require accounts, logins, payment information, contact forms, or user profiles.

GPT Action Data Feed

The Octagon Verdict GPT Action may request a public JSON file from this website to retrieve current fighter ranking data, category scores, comparison notes, and writing guidance. The data feed is public and does not contain personal user information.

ChatGPT Conversations

If you use Octagon Verdict inside ChatGPT, your conversation with ChatGPT is handled by OpenAI and subject to OpenAI's own privacy practices and settings. This website does not receive, store, or process your ChatGPT conversation content.

Cookies and Tracking

This site does not intentionally use cookies, analytics trackers, advertising pixels, or behavioral tracking tools.

Third-Party Hosting

This site is hosted using GitHub Pages. GitHub may process standard technical information such as IP addresses, browser type, and request logs as part of providing hosting services.

Children's Privacy

This site is not designed to collect information from children and does not knowingly collect personal information from children.

Changes to This Policy

This privacy policy may be updated if the site adds new features, data collection, analytics, or account functionality.

Contact

For questions about this privacy policy, contact the site owner through the GitHub repository associated with this project.
